# Skylence Scoop Bucket

Scoop bucket for the [Skylence](https://skylence.be) `sky` CLI — a Claude Code harness builder.

## Install

```powershell
scoop bucket add skylence-be https://github.com/skylence-be/scoop-bucket
scoop install skylence-be/sky
```

## Update

```powershell
scoop update sky
```

Or use the built-in self-updater:

```powershell
sky update
```

## Uninstall

```powershell
scoop uninstall sky
scoop bucket rm skylence-be
```
